# Frontend - VeraGenesi Mobile App

React Native application for iOS and Android.

## Setup

```bash
npm install
npm start
```

## Development

- Run on Android: `npm run android`
- Run on iOS: `npm run ios`
- Run tests: `npm test`

## Project Structure

frontend/
├── src/
│   ├── screens/

│   ├── services/
│   ├── styles/

├── assets/
