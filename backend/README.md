# Backend - VeraGenesi API Server

Node.js Express API server for VeraGenesi.

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file:

```
PORT=3000
NODE_ENV=development
DATABASE_URL=[TBD from spec]
JWT_SECRET=[TBD from spec]
```
